===========
Quick start
===========

General datamesh documentation can be found at https://docs.oceanum.io.

Install
--------

All the examples in this repository are written in Python. 
To run them, you need a minimup of the oceanum python library 
installed (https://oceanum-python.readthedocs.io/en/latest).

    pip install oceanum

If you want to use the jupyerlab extention (https://docs.oceanum.io/datamesh/integrations/jupyterlab).

    pip install oceanumlab
